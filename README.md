# libGDX template

Starter project for libGDX featuring easy asset loading and a scaled pixel-art effect. Perfect for Game Jams.

To get started:

```
$ git clone https://github.com/halfcutdev/libgdx-template
$ cd libgdx-template
$ ./gradlew desktop:run
```
